Tabcast Broadcast and receive tab URL streams

Defining Tabcast groups

Below you can define the Tabcast groups you want to track and/or broadcast URLs to. For more information on using Tabcast in general, visit tabcast.net. Each Tabcast group is specified with the following details:

Nickname
A short memorable name for a group on a Tabcast server. It is just a local name, for your convenience. Group nicknames will be shown in Chrome's context menu.
Server URL
The URL of the Tabcast server for the group. If in doubt, use http://tabcast.net/ as your server. You can also run your own server (Get the Tabcast source from Github).
Group name
The name of the group on the Tabcast server.
User name (Optional)
The user name you wish to use on the server.
Password (Optional)
A password to identify yourself on the server. Note: passwords are stored locally by Chrome, in plaintext, and are therefore not very secure. Do not re-use a password that you use elsewhere.
