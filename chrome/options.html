Tabcast Broadcast and track URLs on a tab-by-tab basis

Groups

Below you can define Tabcast groups that you wish to track and broadcast URLs to. Groups are specified with the following details:

Nickname
A nickname is a short memorable name for a group. You (currently) cannot change it, though groups can be deleted.
Server URL
The URL of the Tabcast server
Group name
The name of the group on the server
User name
Optional. The user name you wish to use on the server.
Password
Optional. A password to identify yourself on the server. Note: passwords are stored locally by Chrome, in plaintext, and are NOT secure.
